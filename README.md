# 🎲 Multiplayer Dice Roller (Jetpack Compose)

A modern **2-player multiplayer dice game** built using **Kotlin** and **Jetpack Compose**.
This project demonstrates clean UI design, state management, animations, and turn-based game logic in Android.

---

## 🚀 Features

✨ 2-Player turn-based gameplay
🎲 Random dice roll with animation
👥 Automatic player switching
🧮 Real-time score tracking
🏆 Winner detection system
🎨 Modern Material 3 UI
⚡ Built entirely with Jetpack Compose

---

## 📱 Demo

> Tap **Roll Dice** and players take turns rolling until someone wins!

---

## 🛠️ Tech Stack

* **Language:** Kotlin
* **UI:** Jetpack Compose
* **Architecture:** Single Activity
* **State Management:** Compose State
* **Animation:** `animateFloatAsState`
* **IDE:** Android Studio

## 🎮 Game Rules

* Player 1 starts first
* Each player rolls the dice on their turn
* The rolled number is added to their score
* First player to reach **50 points** wins 🏆
